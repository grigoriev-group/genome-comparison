### [TE-PB-BB](./TE-PB-BB/)

**Transposable Element Analysis — Polar Bear & Brown Bear**

Code for the structural variant (deletion/insertion) analysis pipeline used in:

> "Recent Dominant Transposition Events Affect Gene Regulatory Regions but Not Coding Sequences in Polar and Brown Bear Genomes" *(in preparation)*

This pipeline identifies and annotates species-specific genomic deletions by cross-species comparison of brown bear (*Ursus arctos*) and polar bear (*Ursus maritimus*) whole-genome sequencing data.

See [`TE-PB-BB/README.md`](./TE-PB-BB/README.md) for full documentation, installation instructions, and usage.
